# Festival Operations Guide

## Customer Experience (Madame Zelda)
- Acknowledge frustration first
- Offer alternatives, never just say "no"
- VIP reading for waits >20 min
- Skeptics → "mysterious stranger" style
- Kids under 10 → "adventure quest" style
- Fortunes max 2–3 minutes
- Extended reading slots for chatty guests
- Keep 5 pre‑written fortunes ready

## Security & Vendor Relations (Marcus)
### Vendor Issues
- Food vendors: check permits before setup
- Noise complaints: verbal → written → shutdown
- Vendor disputes: mediate privately
- Payment: 50% upfront, balance day‑of

### Security Protocols
- Lost child: Code Yellow
- Medical emergency: Code Blue
- Weather emergency: Code White
- Suspicious activity: Code Orange

## Lost & Found (Maria)
### Intake
- Photo everything before storing
- Tag with description, location, time, finder
- High‑value → locked cabinet
- Perishables → dispose after 2 hours

### Matching
- Ask claimant to describe first
- Check ID for high‑value
- Log all claims
- Unclaimed after 30 days → charity

### Common Patterns
- Ice rink → mittens/scarves
- Food court → phones/wallets
- Kids areas → stuffed animals (urgent)

## Marketing & Communications (Elena)
### Last‑Minute Requests
- Poster changes: 2 hours minimum
- Social posts: 15 minutes if content provided
- Press inquiries: route to Director
- Sponsor logos: check contract first

### Emergency Comms
- Weather delays: post to all channels
- Event cancellations: email ticket holders first
- Good news: social → then newsletter
